# Laracasts: Laravel From Scratch (2026)

This directory is a **work-in-progress** Laravel application built while following [Laracasts Laravel From Scratch (2026)](https://laracasts.com/series/laravel-from-scratch-2026).

## Why this exists

Frameworks move quickly. Every so often I like to run through a full, opinionated Laravel path to **stay aligned with current conventions** and **recheck habits** against how the ecosystem expects things to be done today (routing, Blade, Eloquent, migrations, validation, and the rest of the stack as the series covers it). Working in a real project tree beats rereading docs in isolation.

## Status

**In progress.** This README marks a checkpoint; the course is not finished yet and the app may be incomplete relative to the full series.
